## 画像推奨仕様
- 形式: PNG（透過対応）
- サイズ: 横幅800-1200px推奨
- 内容: 
  - コースレイアウト図: 競馬場の全体図・コース形状
  - 距離別データ図: 馬番別勝率・人気別成績などのグラフ

## 配置方法
1. 画像ファイルを `public/course_data/` に配置
2. ファイル名は上記リストと完全一致させる
3. コミット・プッシュで自動反映
